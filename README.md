# DungeonLooter
A script that converts AI Dungeon scenarios to SillyTavern-compatible character card and lore book.

## Usage

1. Download/clone and extract it.
2. Open CMD and run `pip install -r requirements.txt`.
3. Choose a scenario. Copy its ID from the URL. For example, a scenario with URL https://play.aidungeon.com/scenario/CaFeBabe8/cool-scenario has ID `CafeBabe8`.
4. Run the program using `python <id> <character-export-path> [lore-book-export-path]` where the last parameter is optional.
